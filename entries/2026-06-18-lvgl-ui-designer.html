LVGL UI Designer：单片机界面设计技能

普通
LVGL 嵌入式GUI ESP32 STM32 Skill开发 ST7789
◫ LVGL UI Designer

+ 新增技能

  • lvgl-ui-designer — LVGL 单片机界面设计，支持 ESP32/STM32/Arduino 多平台

− 移除技能

认知更新

  • LVGL 9 API 从结构体驱动改为函数式 API：lv_display_create() 替代 lv_disp_drv_register()
  • 显示缓冲区策略：部分缓冲（10-20行）节省内存，双缓冲提升刷新率
  • Flex/Grid 布局比绝对定位更适合响应式嵌入式 UI
  • Catppuccin/Dracula/Nord 等程序员主题可直接用于嵌入式深色界面
  • LVGL 的 Part + State 样式系统类似 CSS 的伪类，支持状态联动

行为变更

  • 基于 LVGL 官方文档构建完整的 Skill，包含模板 + 组件库 + 主题系统
  • 支持多平台（ESP32/STM32/Arduino/Pico）和多显示屏（ST7789/ILI9341/ILI9488）
  • 预设 3 套流行配色主题，可一键切换
  • 可复用组件库：卡片、仪表盘、迷你图表、温度计、状态LED

建立的新习惯

  • 嵌入式 GUI 开发先选主题，再做组件，最后组装界面
  • 使用 Flex/Grid 布局而非硬编码坐标
  • LVGL 项目模板化：显示驱动 + 输入驱动 + UI 层分离

关键教训

  • LVGL 9 是新标准 — 新项目直接用 LVGL 9，API 更简洁，文档更完善
  • 缓冲区策略决定性能 — PSRAM 可以用双缓冲大缓冲区，内部 SRAM 用单缓冲小缓冲区
  • 主题系统提升效率 — 预定义颜色主题 + 可复用样式，避免每个组件重复设置颜色
